impl GitRepo

fn fetch(&self) -> Result<Output, CommandError>

If a repo exists, fetch it. If it doesn't, clone it

This is currently very dumb in the sense that it only checks if a directory exists, not whether it's the git repo represented by self or even whether it's a git repository at all.

This is the equivalent of doing:

(test -d <local_path> && cd <local_path> && git fetch) || \
git clone --depth=1 --single-branch -b <branch> <remote_path> <local_path>

fn checkout(&self, sha: &str) -> Result<Output, CommandError>
